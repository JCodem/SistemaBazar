# Panel de Control - Jefe de Ventas 🧾

Este es un panel de administración desarrollado en React, diseñado para permitir al **jefe de ventas** gestionar de manera eficiente:

- El estado del día (abierto o cerrado)
- El registro de productos
- La visualización de la tabla de productos
- El filtrado de productos por nombre o código
